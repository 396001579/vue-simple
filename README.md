# vue-simple
vue 全家桶模板，支持微信、支付宝网页授权，支付宝、微信支付，支持路由跳转、登录拦截，加入 eslint 校验，git 提交前检测等等，可以直接开发业务代码

 
### use 

```
npm run dev
```
